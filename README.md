   - Command? r: 해당 이름이 발견된 위치(레벨, 레코드 내의 위치) 출력
   - Command? d: 해당 이름을 제거한 다음 성공 여부를 출력(이 과정에서 발생한 redistribution의 총 수, merging의 총 수도 출력)
   - Command? sp: B-tree 안의 모든 레코드의 이름 순으로 SeqData.txt파일에 출력

   - Command? exit: 프로그램 종료
